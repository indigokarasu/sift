# Sift

Web search, research synthesis, fact verification, and entity extraction. Separates signal from noise across the open web. Topic research engine for the OCAS ecosystem.

## Repository Structure

- `ocas-sift/` - Skill package source
- `spec/` - Specification and build documentation

## Installation

This skill package is available on GitHub. To use it, clone this repository and follow the installation instructions in your skill management system.
